# PipeRoute Tabs Privacy Policy

_Last updated: June 2026_

## Overview

PipeRoute Tabs is a Chrome extension that enables the PipeRoute desktop application to discover and route audible Chrome tabs.

## Data Collection

PipeRoute Tabs does **not** collect, store, transmit, or sell personal information.

The extension does not require user accounts and does not send data to external servers.

## Local Communication

The extension communicates only with the locally running PipeRoute desktop application through:

```
http://127.0.0.1:8722
```

This communication never leaves the user's computer.

## Chrome Permissions

The extension requests only the permissions required to:

- detect audible tabs;
- capture audio from the selected tab;
- synchronize tab information with the local PipeRoute application.

## Third Parties

PipeRoute Tabs does not use analytics, advertising, tracking services, or third-party data collection.
